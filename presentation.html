🎯
이 서비스는 무엇인가요?
AI가 회의 안건 입력부터 참석자 추천, 일정 조율, 회의실 예약, 사전 알림, 회의록 작성까지 10단계 전 과정을 자동으로 처리해주는 스마트 보조 에이전트입니다.
🖥️
이 데모는 무엇인가요?
실제 AI 에이전트가 어떻게 생각하고, 어떤 도구를 사용하고, 어떤 결정을 내리는지 눈으로 보고 직접 체험할 수 있는 웹 시뮬레이터입니다.
💡 핵심 아이디어: 사람이 직접 하던 회의 관련 반복 업무를 AI 에이전트에게 위임하여, 실무자는 본질적인 업무에만 집중할 수 있습니다.
AI 에이전트 구성

4개의 AI 에이전트가 역할을 나눠
팀처럼 협력합니다

🧠
회의 조율 에이전트 (총괄)
안건 분석, 참석자 추천, 일정 충돌 감지, 스마트 재조율을 담당하는 중심 에이전트. 나머지 에이전트들에게 업무를 지시합니다.
🏢
자원 예약 에이전트
회의 규모에 맞는 회의실을 자동으로 탐색·예약하거나, 비대면 시 Teams 화상 링크를 생성합니다.
📲
RSVP 관리 에이전트
회의 24시간 전 텔레그램으로 참석 여부를 자동 조사하고, 취소·변경 시 즉시 전체 참가자에게 전파합니다.
✍️
회의록 작성 에이전트
회의 중 실시간 음성을 자동 속기하고, 종료 후 요약·결정사항·Action Item이 포함된 회의록 초안을 즉시 생성합니다.

📌 사람이 하는 일: 회의 안건을 AI에게 말해주는 것, 그리고 최종 선택을 확인하는 것.
나머지 모든 단계는 AI 에이전트가 알아서 처리합니다.

단계별 설명 (1~5단계)

안건 입력부터 참석자 확정까지

1
회의 안건 입력 — "Q3 마케팅 전략 수립 회의, 브랜드 인지도 제고 및 광고 예산 수립" 입력
2
호스트 캘린더 자동 분석 — AI가 내 일정을 보고 비어있는 날짜 3개를 추천 카드로 제시
3
시간대 선택 — 제안된 날짜 중 하나를 클릭하고 선호 시간을 드래그로 설정
4
AI 참석자 자동 추천 — "마케팅·브랜드·예산" 키워드로 인사 DB 검색 → 관련 담당자 자동 매칭
5
일정 충돌 감지 — 필수 참가자들의 캘린더를 실시간 스캔하여 겹치는 일정을 빨간색으로 표시
단계별 설명 (6~10단계)

자원 예약부터 회의록 배포까지

6
일정 스마트 재조율 — 충돌 시 전원이 비어있는 최적 시간대를 AI가 자동으로 재탐색·제안
7
자원 자동 예약 — 대면이면 수용 인원에 맞는 회의실 자동 예약, 비대면이면 Teams 링크 즉시 생성
8
일정 등록 & 알림 발송 — 전 참가자 캘린더에 자동 등록, 이메일·메신저로 안내 발송
9
D-24h 참석 조사 — 텔레그램으로 참석 여부 자동 설문, 취소 시 전원에게 즉시 비상 전파 + 캘린더 취소선
10
회의 진행 & AI 회의록 — 실시간 음성 속기 → 종료 후 요약·결정사항·Action Item 자동 생성 → 피드백 반영 → 전원 공유
데모 체험 포인트

시뮬레이터에서 직접 눈으로 볼 수 있는 것들

🗓️
실시간 캘린더 시각화
호스트와 참석자들의 일정이 겹치는 구간을 빨간색, 여유 시간을 파란색으로 구분해 한눈에 표시
🤖
AI 사고 과정 투명 공개
AI가 "지금 무슨 생각을 하고 있는지", "어떤 도구를 호출했는지", "결과는 무엇인지"를 로그로 실시간 확인
📬
가상 이메일 & 메신저
이메일 발송 애니메이션, 텔레그램 알림 팝업이 실제처럼 동작하는 모의 커뮤니케이션 피드
🎙️
실시간 음성 속기 시뮬레이션
회의 진행 중 텍스트가 실시간으로 스크롤되고, 종료 버튼 클릭 즉시 AI 회의록이 자동 생성
🎨 디자인: 다크 테마 + 글라스모피즘 효과 + 부드러운 애니메이션으로 프리미엄 UI 구현

🧩
ReAct 패턴
AI가 단순히 답을 내놓는 게 아니라
생각(THOUGHT) → 행동(ACTION) → 관찰(OBSERVATION)
의 순환 과정을 거쳐 복잡한 문제를 해결하는 AI 설계 방식
🔧
에이전트 도구(Tools)
AI가 직접 실행할 수 있는 기능들 — 캘린더 조회, 회의실 예약, 이메일 발송 등 각 기능을 AI가 상황에 맞게 선택해 호출합니다.
🤝
멀티 에이전트 협업
하나의 AI가 모든 것을 하는 게 아니라, 전문 AI 4명이 역할을 나눠 각자 잘하는 일을 담당하고 서로 소통하는 구조
🌐
웹 기반 시뮬레이터
별도 설치 없이 브라우저에서 바로 실행 가능. 실제 외부 서비스 연동 없이도 전 과정을 100% 체험할 수 있습니다.
